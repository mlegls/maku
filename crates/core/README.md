# maku

`maku` is the deterministic engine and language package for Maku. It provides
the supported host lifecycle (`maku::host`), embedded source helpers
(`maku::source`), and backend-neutral ordered render transport
(`maku::render`).

Genre rendering and delivery hosts are separate packages. Use
`maku-render-touhou` for the bundled Touhou-style render profile,
`maku-player` for the native reference player, or `maku-web` for wasm host
bindings.

See the [repository](https://github.com/mlegls/maku) for language tutorials,
capability specifications, and complete examples.
